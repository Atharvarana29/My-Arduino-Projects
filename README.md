## 🚀 Projects Included

### 🔹 Line Follower Robots (LFR)

* **5-Channel Non-PID LFR**

  * Basic line following using IR sensor array
  * Logic-based movement control (no feedback tuning)

* **5-Channel PID LFR**

  * Advanced line following using PID control
  * Improved accuracy, stability, and smooth motion

---

### 🔹 Sensor-Based Systems

* **Gas Sensor System**

  * Detects presence of harmful gases
  * Can be extended for safety/alert systems

* **Vibration Detector**

  * Detects motion or vibration
  * Useful for security and monitoring applications

---

### 🔹 Embedded Display Project

* **Walkman Simulator with OLED Display**

  * Simulates a music player interface
  * Uses OLED for output visualization
  * Demonstrates UI + embedded integration
